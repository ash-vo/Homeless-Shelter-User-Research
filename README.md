# Homeless Shelter App – User Research

## Overview
This project explores how individuals experiencing homelessness find and access shelter resources. The goal was to identify barriers and design a more accessible solution.

## My Role
- Conducted user interviews  
- Synthesized research findings  
- Contributed to design decisions  

## Methods
- User interviews  
- Thematic analysis  

## Key Insights
- Users struggled to find up-to-date shelter availability  
- Navigation and accessibility were major barriers  
- Trust and reliability of information were critical  

## Outcome
Designed a concept for a mobile app that improves access to shelter information through clearer navigation and real-time updates.
